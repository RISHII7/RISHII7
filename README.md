* 🌱 **Expanding Expertise:** Deep-diving into Next.js 15 routing, React 19 server actions, and advanced JavaScript design patterns.
* 💬 **Ask Me About:** Full-stack architecture, database design (Prisma, PostgreSQL, Mongo), and building serverless platforms like **Roomify**, **Finance Tracker**, and **YOOM**.
